# MVP Roadmap

## Phase 1: Foundation

- React dashboard shell
- Express orchestration API
- deterministic mock workflow
- initial Move contract skeletons
- project docs and pitch framing

## Phase 2: Live workflow loop

- connect frontend task form to backend
- render real task execution responses
- persist demo tasks
- add loading, error, and completed states

## Phase 3: AI integration

- configure Gemini API
- improve agent prompts
- add structured JSON parsing
- add validation score contract between agents

## Phase 4: Sui integration

- publish task manager contract
- create task objects from wallet transactions
- emit workflow events
- connect transaction digests to the dashboard

## Phase 5: Walrus integration

- upload reports and logs
- store blob IDs on task records
- expose artifact retrieval in dashboard

## Phase 6: Economic proof

- add escrow contract calls
- release funds after validation threshold
- update agent reputation after settlement
- prepare final demo script
